# Windows XP Web Clone

A faithful recreation of the classic Windows XP desktop environment, built entirely with HTML, CSS, and JavaScript. This project aims to bring the nostalgia of the early 2000s to the modern web.

## üåü Features

- **Authentic Desktop Experience**:
  - Classic desktop icons (My Computer, Recycle Bin, Internet Explorer).
  - Drag-and-drop support for icons.
  - Context menus (Right-click) on desktop and files.
  - "Bliss" wallpaper (or classic blue background).

- **Window Management System**:
  - Fully functional windowing system (Open, Close, Minimize, Maximize).
  - Draggable and resizable windows.
  - Taskbar integration with active application states.

- **Start Menu**:
  - accurate recreation of the specific Windows XP Start Menu layout.
  - Nested menus and quick links.
  - Functional "Turn Off / Log Off" dialogs.

- **Included Applications**:
  - **Internet Explorer**: A mockup of the classic browser.
  - **Notepad**: A working text editor that saves to LocalStorage.
  - **My Computer**: Navigate through virtual drives and folders.
  - **Recycle Bin**: Delete and restore items.
  - **Calculator**: Basic arithmetic operations.
  - **Minesweeper**: The classic puzzle game.
  - **Media Player**: Basic music playback capabilities.

- **System Tray**:
  - Live digital clock.
  - Volume control.
  - Network and Security status indicators.

## üöÄ Getting Started

### Prerequisites
You need a modern web browser (Chrome, Firefox, Edge, Safari) to run this project. No server or backend is required.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/skdsam/windowsxp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd windowsxp
   ```
3. Open `index.html` in your preferred web browser.
